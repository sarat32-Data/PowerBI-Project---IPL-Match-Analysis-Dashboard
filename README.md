# PowerBI-Project---IPL-Match-Analysis-Dashboard
Just built an IPL Match Analysis Dashboard in Power BI — and the numbers are fascinating!  I've always believed cricket is not just a sport — it's a data goldmine. So I decided to dig deep into IPL match data and turn raw numbers into visual stories.
📊 Here's what the dashboard covers:

✅ KPI Cards
→ Total Matches played across all seasons
→ Total Seasons analyzed
→ Avg First Innings Score
→ Avg Second Innings Score
→ Toss Win → Match Win %

✅ Team Performance Analysis
→ Total Wins Per Team (all-time)
→ Teams Win Per Season (trend over years)
→ Top Team — Most Wins All Time
→ Highest Avg Winning Score & the team behind it
→ Win Distribution — Top 5 Teams

✅ Strategy Insights
→ Bowl First vs Bat First — which decision wins more?
→ Toss impact on match results

💡 Key insight that surprised me:
The Toss Win → Match Win % reveals just how much a coin flip can influence a professional cricket match outcome. Sometimes strategy begins even before the first ball is bowled.

🛠 Tools used: Power BI | DAX | Data Modeling

This project sharpened my skills in:
→ Writing complex DAX measures
→ Designing clean, story-driven dashboards
→ Translating sports data into business-style insights
